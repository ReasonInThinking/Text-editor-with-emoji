# Emoji Message Input & Picker

A lightweight, purely vanilla HTML/JS/CSS tool that provides an interactive message input field with a categorized emoji picker and a smart clipboard copy feature.

## 🚀 Features

- **Categorized Emoji Picker:** Choose from 5 different sets of emojis (Smiles, Home, Electronics, Strategy, and Activity).
- **Smooth Interaction:** Only one category is shown at a time; opening a new one automatically closes the previous one.
- **Message Input:** A custom-styled `textarea` that accepts both text and emojis.
- **One-Click Copy:** A dedicated button to copy the entire message to your clipboard.
- **Copy Notification:** Users receive a confirmation alert when the text is successfully copied.
- **No Dependencies:** Built with pure HTML, CSS, and JavaScript. No external libraries required.

## 🛠️ Technology Stack

- **HTML5:** Semantic structure and emoji entities.
- **CSS3:** Custom layout with responsive `textarea` and hidden/visible state management.
- **JavaScript (ES6):** DOM manipulation, state toggling, and Clipboard API.

## 📋 How to Use

1. **Select Category:** Click on one of the top icons (😀, 🏠, 🔌, 📌, 🏇) to open an emoji list.
2. **Add Emojis:** Click on any emoji in the list to instantly append it to the text area.
3. **Write:** You can type additional text manually in the box.
4. **Copy:** Click the "Copy Text" button to grab your message. An alert will confirm the action.
